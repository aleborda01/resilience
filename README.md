# AI Resilience Accelerator

Landing page for the AI Resilience Accelerator — a 14-week program helping AI researchers become founders building civilization-scale solutions for AI safety and resilience.

## About

A partnership between OpenAI Foundation and Fifty Years, this accelerator takes the proven 5050 model (340+ scientists trained, 95% seed success rate) and applies it to AI resilience challenges including:

- Alignment & interpretability
- AI governance & policy
- Human-AI interfaces
- Biosecurity & countermeasures
- AI-powered cybersecurity

## Deployment

This site is deployed via GitHub Pages. Simply push to the `main` branch and enable Pages in repository settings.

## Local Development

Open `index.html` in any browser — no build step required.
